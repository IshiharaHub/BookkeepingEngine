# BookkeepingEngine

簿記の計算を自動でしてくれる WEB アプリ

- https://ishiharahub.github.io/BookkeepingEngine/src/index.html

## ファイル変更時の操作方法

### index.html を修正した場合

1. ファイルを修正する
2. 修正したファイルをステージングに追加する `git add index.html`
3. 修正したファイルを commit する `git commit -m"ページのヘッダーを修正"`
4. 修正したファイルを push する `git push`

## 画面一覧

- 計算ツール
  - 総合原価計算
    - 平均法
    - 先入先出法
  - 減価償却計算
  - 繰延税金計算
  - 利益分析（ブレークイーブン点計算）
  - キャッシュフロー予測
  - 在庫評価
- その他
  - フィードバック
  - コンタクト
  - プライバシーポリシー
